## Análisis para tienda Ice
La tienda online Ice vende videojuegos por todo el mundo. Las reseñas de usuarios y expertos, los géneros, las plataformas (por ejemplo, Xbox o PlayStation) y los datos históricos sobre las ventas de juegos están disponibles en fuentes abiertas. Se realiza un análisis para identificar patrones que determinen si un juego tiene éxito o no.
